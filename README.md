# Token Visualizer

An interactive tool for exploring LLM tokenization, comparing models, and estimating API costs.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- js-tiktoken

## Getting Started

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

The app will be available at `http://localhost:8080`.
